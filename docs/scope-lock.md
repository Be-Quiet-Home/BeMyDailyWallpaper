BeMyDailyWall is a small Haiku-native daily wallpaper companion.

It is not:
- a cloud photo service
- a social image platform
- a Qt application
- a browser-based wallpaper app
- a branded Bing/NASA client

It is:
- a Deskbar companion
- a daily wallpaper fetcher
- a metadata-aware image archive
- a small window to somewhere else
